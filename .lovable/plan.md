
# Cayley Protocol — Landing Page

## Overview
A dark, techy landing page for Cayley Protocol — a multichain decentralized slice indexing protocol. Clean dark backgrounds, neon/cyan accents, grid patterns, and sharp typography.

## Design System
- **Background**: Deep dark navy/black (#0a0e1a style)
- **Accent**: Cyan/electric blue glow
- **Typography**: Clean sans-serif, bold headlines
- **Effects**: Subtle grid pattern background, glowing accents, gradient borders

## Sections

### 1. Navbar
- "Cayley" logo/wordmark on the left
- Nav links: Features, How It Works, Docs (placeholder)
- "Launch App" CTA button with glow effect

### 2. Hero
- Bold headline: "Multichain Decentralized Slice Indexing"
- Subtitle explaining the protocol's value prop
- Two CTAs: "Get Started" and "Read Docs"
- Abstract geometric/grid visual element

### 3. Features Grid
- 4 feature cards with icons in a responsive grid:
  - **Multichain Support** — Index across multiple blockchains
  - **Decentralized Indexing** — No single point of failure
  - **Slice Architecture** — Efficient data partitioning
  - **Developer-First** — Simple APIs and SDKs

### 4. Footer
- Cayley Protocol branding
- Links to social placeholders (Twitter, Discord, GitHub)
